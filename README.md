# Vinheria Agnello

Site institucional da Vinheria Agnello — catálogo de vinhos, galeria, equipe e contato.

## Estrutura

```
├── index.html
├── src/pages/
│   ├── produto.html       (catálogo + tabela harmonização)
│   ├── galeria.html       (galeria de imagens)
│   ├── equipe.html        (informações equipe)
│   └── contato.html       (formulário contato)
├── style.css   
│ └── efeitos.css
└── src

```

# Tecnologias

- CSS
